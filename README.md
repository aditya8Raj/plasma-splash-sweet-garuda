# Installation
# Arch Linux

Install the package <code>plasma-splash-sweet-garuda-git</code><br>
```
$ yay -S plasma-splash-sweet-garuda-git
```

# Other Distros
Copy the folder Sweet-Garuda to ~/.local/share/plasma/look-and-feel/

# Set the Splash Screen
